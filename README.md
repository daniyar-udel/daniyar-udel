## Featured Projects

### TreeRoute
Hackathon-winning multimodal walking route planner for allergy-sensitive New Yorkers.  
Combines NYC tree census data, live pollen conditions, live weather/wind, Google Maps routing, and Gemini-generated grounded explanations to rank walking routes by likely pollen exposure.

**Tech:** Next.js, TypeScript, Google Maps API, Google Routes API, Google Pollen API, Google Weather API, Gemini, Docker, Cloud Run  
[Repository](https://github.com/daniyar-udel/treeroute)

### Knowledge Copilot
End-to-end RAG-style system for grounded Q&A over PDF documents.  
Supports upload, indexing, semantic retrieval, streaming answers, citations, explicit user feedback, and lightweight analytics.

**Tech:** FastAPI, LangChain, Chroma, Ollama, Next.js, React, TypeScript  
[Repository](https://github.com/daniyar-udel/knowledge-copilot)

### AI Investment Strategy Advisor
Full-stack, market-aware investment planning system powered by ML and LLMs.  
Classifies user risk profile, adapts allocations by market regime, runs Monte Carlo simulations, and generates plain-English recommendations.

**Tech:** Python, FastAPI, scikit-learn, LangGraph, LangChain, Groq, React, TypeScript  
[Repository](https://github.com/daniyar-udel/Financial-Planning-Advisor)
